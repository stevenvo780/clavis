## 1. Delimitación conceptual: ¿de qué hablamos cuando hablamos de conciencia?

Antes de teorizar, hay que distinguir. Muchas veces confundimos el fenómeno de la conciencia con otras cosas.

### Lo que NO es conciencia

| Término                   | Definición                                                                                                                                                    |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Auto-consciencia**      | Conciencia de los propios estados mentales; estar al tanto de los propios estados mentales, juicios o creencias sobre estados mentales propios.               |
| **Conciencia moral**      | Discriminación entre lo bueno y lo malo. Es otra cosa totalmente distinta.                                                                                    |
| **Vigilia (wakefulness)** | Ciclo sueño-vigilia, reactividad. Disociable de la conciencia (Laureys — pacientes en estado vegetativo pueden tener ciclos de sueño/vigilia sin conciencia). |

> 📝 *Cita de clase*: «Dichoso el árbol, que es apenas sensitivo, mas la piedra dura porque esa ya no siente...» (Rubén Darío, *Lo fatal*). El profe la usó para ilustrar que la sensibilidad y la experiencia son el núcleo de lo que nos importa.

### Lo que SÍ es conciencia (para este curso)

```
┌────────────────────────────────────────────────────────────┐
│                   TIPOS DE CONCIENCIA                       │
├─────────────────────┬──────────────────────────────────────┤
│ CONCIENCIA          │ Qualia: el "qué se siente"           │
│ FENOMÉNICA          │ Rojez del tomate, brillo de la luz,  │
│                     │ frío, calor, dolor, sabor            │
│                     │ → Carácter subjetivo de la experiencia│
│                     │ → Estados fenoménicos paradigmáticos  │
├─────────────────────┼──────────────────────────────────────┤
│ CONCIENCIA          │ Información disponible globalmente   │
│ DE ACCESO           │ para reporte verbal, razonamiento    │
│ (Block, 1995)       │ y control de la acción               │
└─────────────────────┴──────────────────────────────────────┘
```

Todas estas acciones corresponden a nuestras modalidades específicas sensoriales. De ahí surge la idea de que hay un tipo de experiencia que tienen otros seres y que tiene un **carácter subjetivo**: el mundo se me aparece de cierta manera, se nos aparece de forma fenomenológica.

## 2. La asimetría epistémica

```
        MUNDO OBJETIVO                    MUNDO SUBJETIVO
   ┌──────────────────┐            ┌──────────────────┐
   │  Objetos, cosas  │            │  Experiencias     │
   │  Entidades que   │            │  conscientes      │
   │  aparecen "ahí"  │            │  Solo accesibles  │
   │  Estudiables en  │            │  en 1ª persona    │
   │  3ª persona      │            │                   │
   └──────────────────┘            └──────────────────┘
            │                              │
            └──────────┬───────────────────┘
                       │
              ASIMETRÍA EPISTÉMICA
         Solo YO sé cómo se siente MI
         experiencia fenoménica particular.
         La ciencia solo puede estudiar
         la conciencia A TRAVÉS de nuestras
         propias experiencias.
```

No hay forma de acceder a la conciencia independientemente de nuestras formas de acceso. Estas constituyen la conciencia misma.

## 3. El Problema Difícil de la Conciencia (Chalmers, 1995)

### La pregunta

Cuando cerramos los ojos, cortamos el flujo de información que viaja a través de la retina, nervio óptico, etc., y quedamos "en negro". Pero cuando abrimos los ojos, la fenomenología perceptual es muy rica: podemos ver un paisaje completo.

**La pregunta**: ¿cómo es posible que esta riqueza fenoménica resida en un órgano como el cerebro —un órgano encerrado en el cráneo, que se comunica esencialmente por nervios?

> El problema difícil de la conciencia: ¿cómo el procesamiento físico de información puede generar una experiencia rica y fenomenológica del mundo?

### Problemas fáciles vs. Problema difícil

```
┌─────────────────────────────────────────────────────────────────┐
│               PROBLEMAS FÁCILES                                  │
│  (resolubles con ciencia normal)                                 │
│  · Discriminación sensorial      · Integración de información    │
│  · Reporte verbal                · Control motor                │
│  · Atención                      · Vigilia                      │
├─────────────────────────────────────────────────────────────────┤
│               PROBLEMA DIFÍCIL                                   │
│  ¿Por qué todo ese procesamiento va acompañado de UN "QUÉ SE     │
│  SIENTE"? ¿Por qué hay experiencia subjetiva en absoluto?        │
│  → Podría requerir NUEVOS PRINCIPIOS FUNDAMENTALES               │
└─────────────────────────────────────────────────────────────────┘
```

### La brecha explicativa (Levine, 1983)

Por más neurociencia que tengamos, hay un salto inexplicado entre:

* **Lado objetivo**: procesamiento físico de información en el cerebro
* **Lado subjetivo**: la riqueza fenoménica de la experiencia

> «No conocemos ni podemos conocer la propiedad P que explica la relación entre la mente y el cerebro. Por más neurociencia que hagamos, explicar realmente la conciencia es un problema en sí mismo.»

Mientras tengamos los fenómenos conscientes —que son los que nos constituyen como seres subjetivos— el problema parece irresoluble.

### Argumento de Mary, la neurocientífica (Jackson, 1986)

```
┌──────────────────────────────────────────────────────────┐
│  MARY sabe TODO sobre la percepción del color:            │
│  · Neurología completa del sistema visual                 │
│  · Longitudes de onda, procesamiento retinal              │
│  · V1, V2, V4, áreas corticales                           │
│  · Toda la información física sobre el color              │
│                                                          │
│  Pero Mary NUNCA ha experimentado el color.               │
│  Cuando sale de su cuarto blanco y negro y ve el rojo     │
│  por primera vez... ¿aprende algo NUEVO?                  │
│                                                          │
│  → Si la respuesta es SÍ, entonces hay algo en el mundo   │
│    que NO es información física: las qualia.              │
└──────────────────────────────────────────────────────────┘
```

Si las experiencias fenoménicas son diferentes a toda la información física, entonces hay algo en el mundo que no es físico: las experiencias conscientes, las **qualia**. Por más que estudiemos el cerebro, no podemos entender las qualia solo desde lo físico.

## 4. Qualia y experiencia consciente

La experiencia consciente tiene un **carácter cualitativo**: aquello que hace que X se sienta como X y no como Y. Que hace que el rojo se sienta como rojo y no como verde. Color, textura, sabor, dolor, frío...

No hay duda de que la experiencia consciente está estrechamente asociada con procesos físicos en sistemas como el cerebro. Estos procesos parecen dar lugar a la experiencia. Pero la pregunta de **cómo y por qué** un sistema físico genera experiencia —y por qué la genera— no está clara.

**Ese es el misterio central.**

## 5. Correlatos Neuronales de la Conciencia (NCC)

### Definiciones clave

| Sigla    | Definición                                                                                                   |
| -------- | ------------------------------------------------------------------------------------------------------------ |
| **NCC**  | Correlatos Neuronales de la Conciencia: el estado neuronal mínimo suficiente para una experiencia consciente |
| **NCCs** | NCC de *estado*: suficiente para un contenido específico (ver rojo, oler humo)                               |
| **NCCc** | NCC de *criatura*: suficiente para estar despierto y alerta                                                  |
| ⚠️       | Los NCCs y NCCc se solapan                                                                                   |

### ¿Es V1 necesaria y suficiente para la conciencia?

```
PREGUNTA INICIAL:
¿La corteza visual primaria (V1) es necesaria y suficiente
para la conciencia visual?

┌─────────────────────────────────────────────────────────┐
│                   EVIDENCIA                              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  VISIÓN CIEGA (Blindsight)                              │
│  ┌─────────────────────────────────────────┐            │
│  │ Pacientes con lesión en V1.              │            │
│  │ Campo visual ciego.                      │            │
│  │ Si les muestran un vaso de agua:         │            │
│  │   · Reportan: "NO VEO nada"              │            │
│  │   · Pero... si les piden que lo tomen,   │            │
│  │     ¡LO HACEN correctamente!             │            │

│  └─────────────────────────────────────────┘            │
│                                                         │
│  → V1 parece NECESARIA para la conciencia visual        │
│    (cuando está lesionada, no hay conciencia)           │
│                                                         │
│  EXPERIMENTO DE RIVALIDAD BINOCULAR EN MONOS             │
│  ┌─────────────────────────────────────────┐            │
│  │ 2 monos, estímulo diferente en cada ojo. │            │
│  │ Cuando un ojo tiene mucho ruido, no ven  │            │

```

### El límite del correlato

Identificar correlatos (actividad en V1, región frontoparietal, etc.) tiene una limitación crítica:

> **El dónde no explica el cómo.** Mapea la geografía pero no explica la maquinaria.

## 6. Teorías principales de la conciencia

### 6.1 Teoría del Espacio de Trabajo Global (GWT)

> Baars (1988) · Dehaene (2001+)

**Idea central**: La conciencia es **información globalmente accesible**. Un contenido se vuelve consciente cuando se "transmite" a través de un espacio de trabajo neuronal distribuido (corteza prefrontal, parietal, cíngulo).

```
        ESTÍMULO                     ESTÍMULO
      ENMASCARADO                  CONSCIENTE
   ┌──────────────┐           ┌──────────────────┐
   │  Activación  │           │   IGNICIÓN        │
   │  localizada  │           │   CORTICAL        │
   │  (pocas      │           │                   │
   │   áreas)     │           │  Activación amplia│
   │              │           │  en red fronto-   │
   │  No alcanza  │           │  parietal         │
   │  el umbral   │           │                   │
   │              │           │  Accesible para:  │
   │  Sin reporte │           │  · Memoria        │
   │  Sin recuerdo│           │  · Lenguaje       │
   └──────────────┘           │  · Planificación  │
                              │  · Control motor  │
                              └──────────────────┘
```

**Firma neuronal (Dehaene)**:

* Estímulo consciente → activación tardía (~300 ms) distribuida en redes frontoparietales
* Estímulo no consciente → activación localizada que no se propaga

**Línea temporal de la ignición**:

```
  0 ms        100 ms          200 ms             300 ms
  │            │               │                  │
  Estímulo    Umbral mínimo   Propagación        Estado de
              de ignición     cortical           conciencia
```

**Fuerza del estímulo**: Estímulos más intensos producen activación más amplia (de corteza occipital a temporal). Cuando hay enmascaramiento, se impide la transmisión al resto del cerebro → el paciente no puede recordarlo aunque haya prestado atención.

**La conciencia NO es gradual**: sucede o no sucede (fenómeno todo-o-nada). La ignición es un evento discreto.

**Límite del GWT**: Explica muy bien la conciencia de acceso (información reportable). Pero **no resuelve el problema difícil**: ¿por qué la información globalmente accesible *se siente* de cierta manera?

***

### 6.2 Teoría del Procesamiento Recurrente (Lamme)

**Idea central**: La información NO tiene que propagarse globalmente para ser consciente. Puede ser consciente mediante procesamiento local recurrente.

```
   MODELO FEED-FORWARD SWEEP + PROCESAMIENTO RECURRENTE

   Primer barrido (feed-forward sweep):
   ┌──────┐    ┌──────┐    ┌──────┐    ┌──────┐
   │ V1   │───→│ V2   │───→│ V3   │───→│ TE   │  → RESPUESTA
   └──────┘    └──────┘    └──────┘    └──────┘    MOTORA
       ↑          │            │            │
       │          │            │            │
       └──────────┴────────────┴────────────┘
            PROCESAMIENTO RECURRENTE (feedback)
            → Esto produce la CONCIENCIA VISUAL
              (entre 100 ms en adelante)
```

**Punto clave**: La experiencia visual consciente surge cuando la información procesada en áreas visuales superiores REGRESA a las áreas visuales inferiores (incluyendo V1).

**Evidencia**: Sujetos expertos pueden ser influenciados por palabras enmascaradas aunque digan no verlas (priming semántico sin conciencia). Ese primer barrido feed-forward genera respuestas motoras y semánticas sin necesidad de conciencia.

> La tesis del **inconsciente cognitivo**: una serie de procesos cognitivos que se pueden hacer bien pero sin ser conscientes. El sistema que nos permite actuar "como zombies" sin poner demasiada atención.

***

### 6.3 Comparación GWT vs. Procesamiento Recurrente

```
┌────────────────────────────────────────────────────────────────┐
│           GWT (Dehaene)          │  PROCESAMIENTO RECURRENTE    │
│                                  │  (Lamme)                     │
├──────────────────────────────────┼──────────────────────────────┤
│ Conciencia = información         │ Conciencia = procesamiento   │
│ globalmente transmitida          │ recurrente local             │
├──────────────────────────────────┼──────────────────────────────┤
│ Solo lo atendido es consciente   │ Lo consciente es MÁS AMPLIO  │
│                                  │ que lo atendido               │
├──────────────────────────────────┼──────────────────────────────┤
│ Ventaja: captura idea de sentido │ Ventaja: observa que parece  │
│ común (lo consciente se reporta) │ haber más contenido          │
│                                  │ consciente del que se reporta│
└──────────────────────────────────┴──────────────────────────────┘
```

## 7. Paradigmas experimentales clave

### 7.1 Paradigma de Sperling (memoria icónica)

```
   Fase 1: Presentación            Fase 2: Tono
   ┌─────────────────┐              
   │ A  B  C  D      │             Tono AGUDO  → reportar fila 1
   │ E  F  G  H      │  50 ms      Tono MEDIO  → reportar fila 2
   │ I  J  K  L      │             Tono GRAVE  → reportar fila 3
   └─────────────────┘              
                                    Resultado:
   Sin tono: reportan ~35%         Con tono: reportan CASI TODA
                                    la fila indicada → ¡eran
                                    conscientes de todo!
```

**Conclusión**: La información SÍ era consciente incluso sin ser reportada. La conciencia es más amplia que el reporte.

### 7.2 Paradigma de cambio con máscara (ceguera inatencional)

```
    Fijación      Estímulo      Máscara        Flecha
      │          (500 ms)    (100-1500 ms)   (¿cambió?)
      │              │            │              │
      ▼              ▼            ▼              ▼
   ┌──────┐    ┌──────────┐ ┌──────────┐   ┌─────────┐
   │  ·   │    │  Figura  │ │  Máscara │   │ ¿Cambió │
   └──────┘    └──────────┘ └──────────┘   │ la       │
                                           │ figura?  │
                                           └─────────┘
   SIN flecha durante la máscara → CEGUERA INATENCIONAL
   CON flecha durante la máscara → EL PACIENTE COMPARA
   los dos estímulos → la información persistía en los
   córtex visuales.
```

Si la información NO hubiera sido consciente, no habría constancia entre las dos presentaciones. El experimento se ha replicado con muchos tipos de estímulos y funciona consistentemente bien.

## 8. Conciencia fenoménica vs. Conciencia de acceso (Ned Block)

No es lo mismo **cómo se siente** (fenoménica) que **darse cuenta** (acceso).

```
┌─────────────────────────────────────────────────────────┐
│  CONCIENCIA FENOMÉNICA           CONCIENCIA DE ACCESO   │
│  ┌─────────────────────┐        ┌─────────────────────┐ │
│  │ Qualia, experiencia  │        │ Información         │ │
│  │ subjetiva pura       │        │ disponible para:    │ │
│  │ "qué se siente"      │        │ · Reporte verbal    │ │
│  │                       │        │ · Razonamiento      │ │
│  │ No necesariamente     │        │ · Guía de acción   │ │
│  │ reportable            │        │                     │ │
│  └─────────────────────┘        └─────────────────────┘ │
│                                                         │
│  Las teorías actuales (GWT, recurrente) explican         │
│  principalmente la conciencia DE ACCESO.                 │
│  La conciencia FENOMÉNICA sigue siendo esquiva.          │
│  Ahí es donde hay algunos avances, pero limitados.       │
└─────────────────────────────────────────────────────────┘
```
